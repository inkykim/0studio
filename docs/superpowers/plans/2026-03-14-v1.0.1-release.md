# v1.0.1 Release Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v1.0.1 — a Homebrew-distributable macOS build with cloud sync for single users, team/collaboration and payment features hidden behind compile-time flags.

**Architecture:** Compile-time feature flags via Vite env vars (`VITE_FEATURES_TEAM`, `VITE_FEATURES_PAYMENTS`). Components import `features` from `src/lib/features.ts` and conditionally render. Backend deployed to Railway with CORS updated for Electron's `file://` origin.

**Tech Stack:** Vite, React, Electron, Express, Railway, Homebrew Cask

**Spec:** `docs/superpowers/specs/2026-03-14-v1.0.1-release-design.md`
